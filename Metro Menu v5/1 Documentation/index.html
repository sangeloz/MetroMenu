Installation

 

What do I need to get this into my site?

Add the following in the <HEAD> section

 

First, The jQuery version. (This is not an special one, if you use any jQuery version you don't need to import it again)

<script src="static/js/jquery-1.11.0.min.js"></script>
*Works with 1.6 or later.
 
 
 
Next... The MetroMenu Style
<link href="static/css/MetroMenu.css" rel="stylesheet" type="text/css" media="screen" />
* This is what makes the style
 
 
 
MetroMetu JavaScript
<script src="static/js/MetroMenu.js"></script>

*This is what do the magic.

 
 
Import the Font-Awesome.min.css  (all this files are included in this package)
<link rel="stylesheet" href="static/css/font-awesome.min.css">
 

Properties

 

animation:

* There are 14 animations

 

  1. fadeIn
  2. fadeInRight
  3. fadeInLeft
  4. bounceInLeft
  5. flipInX
  6. flipInY
  7. fadeInUpBig
  8. fadeInDownBig
  9. fadeInLeftBig
  10. fadeInRightBig
  11. bounceIn
  12. bounceInUp
  13. bounceInDown
  14. bounceInRight
 

**Speed up the animation:

Just add the word "fast" to speed up the animation. For example

  1. fadeIn fast
  2. fadeInRight fast
  3. fadeInLeft fast
  4. bounceInLeft fast
  5. flipInX fast
  6. flipInY fast
  7. fadeInUpBig fast
  8. fadeInDownBig fast
  9. fadeInLeftBig fast
  10. fadeInRightBig fast
  11. bounceIn fast
  12. bounceInUp fast
  13. bounceInDown fast
  14. bounceInRight fast
 

position:

* 4 positions

  1. left
  2. right
  3. top
  4. bottom
 

withtooltip:

*Do you want the tool tip or not

  1. true
  2. false

 

backicon: 

* You can set you custom back arrow icon. Example: "static/img/back". This image will appear when you are in a submenu

 

backfa:

Font-Awesome icon to indicate that you can return to the main menu. just pick the icon you want and type it like this: "fa-arrow-left fa-2x"

 

backlabel

Label for the BackIcon, this is the label that will be under the backicon image. (this is optional)

 
 

closeonclick:

* Set this value to true, if you want to close immediately after pressing a option (Except it has submenu items)

  1. true
  2. false

 

escclose:

*Set this value to TRUE, if you want to close the menu by pressing ESC buttom

  1. true
  2. false

 

divholderid:

This is the ID of the container that you whant to store the Metro Menu, this is for using metro menu inside a section instead a fixed position. Just enter the ID like: "#divMyCustomDiv"

 

color1:

Main color... This requires an HTML color. Example "#95b201",

 

color2:

Color that appears on mouse hover. Also requireds an HTML color  Example: "#8aa404",

 

Every menu option has properties that let you change the behavior of the menu, I will highly recommend you to check the tutorials and test it for your selft.

 

Items:

This is an Array that holds the menu options. Iside this you can add the following properties:

  1. name:  Label for the Menu Option
  2. icon: Like "static/img/image1.png", Image URL of the menu option
  3. fa:  Font-Awesome icon... like:  "fa-bullhorn" but you can add special font-awesome functions like: "fa-bullhorn fa-2x fa-spin"
  4. move: If you want to autostroll to a specifit section of yoyr page, just add the element ID like "#divMain" or type "bottom" or "top"
  5. time: Time to get to a specifict element (needs the move property)
  6. link: This creates a menu option with an anchor that goes to this url. 
  7. target: You can add "blank", if you want to open a link in another page
  8. Items: This is an array that holds another submenu items. You can add till 3 levels. Check the tutorial for a examples.
